# Multi-Modal Biometric Authentication System

A Flutter-based Android application implementing:

- Face Recognition using MobileFaceNet
- Voice Biometric Authentication
- Device Biometric (Fingerprint)
- Alexa-style Text-to-Speech feedback

## Features

- Real-time face recognition
- On-device ML processing
- Voice authentication
- Cosine similarity matching
- Text-to-Speech greeting
- Clean architecture
- Offline functionality

---

## Tech Stack

- Flutter
- TensorFlow Lite
- Google ML Kit
- Provider (State Management)
- Flutter TTS
- Local Auth
